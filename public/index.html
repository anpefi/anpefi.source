Redesigning the website

Dec 20, 2017 1 min read
I’ve changed the website design. For the last 2 years this site was relying on the Jekyll framework using the Minimal Mistakes template. But recently I started to dive deeper in Rstudio, rmarkdown and the tidyverse ecosystem. Few days ago I discovered in Twitter about the new Yihui Xie’s package called blogdown that allows to create static websites using Rmarkdown and Hugo. After a quick read of the Xie’s book about this package I decided to change this website from Markdown+Jekyll to Rmarkdown+Blogdown+Hugo.

msap: A tool for the statistical analysis of methylation-sensitive amplified polymorphism data

In this paper I presented an R package which analyses methylation-sensitive amplified polymorphism (MSAP or MS-AFLP) data. The program provides a deep analysis of epigenetic variation starting from a binary data matrix indicating the banding pattern between the isoesquizomeric endonucleases HpaII and MspI, with differential sensitivity to cytosine methylation. The package provides, in a user-friendly command line interface, a pipeline of different analyses of the variation (genetic and epigenetic) among user-defined groups of samples, as well as the classification of the methylation occurrences in those groups.
Andrés Pérez-Figueroa
Mol Ecol Res, 2013.
DOI:10.1111/1755-0998.12064
